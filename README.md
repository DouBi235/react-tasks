# React Crash Course 2021 (Task Tracker App)

## Usage

### Install dependencies

```
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the JSON server (http://localhost:5678

```
npm run server
```

### To build for production

```
npm run build
